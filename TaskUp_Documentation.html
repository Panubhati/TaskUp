AI-Powered Competitive Coding & Talent Assessment Platform
Full-Stack Web Application — Project Documentation v1.0

1. Project Overview

TaskUp is a full-stack, AI-powered competitive coding and talent assessment platform that connects students (developers) with companies (recruiters). Students solve real-world coding challenges, take quizzes, complete assignments, and get ranked on live leaderboards. Companies create challenges, evaluate submissions with AI-driven analysis, and identify top talent.

🎓 For Students

Solve challenges in 4 languages, get AI feedback, compete on leaderboards, AI resume evaluation

🏢 For Companies

Create tasks/assignments/quizzes, AI-evaluated submissions with plagiarism detection, analytics dashboard

5. Database Models

User Model

FieldTypeDescription
usernameString (unique)Login username
passwordStringBcrypt-hashed password
roleEnum: student/company/adminUser role

email, phone, dob, college, city, qualifyingYearVariousStudent profile fields
companyName, industry, companySize, website, companyDescriptionVariousCompany profile fields

Task Model

Contains title, description, author (company), examples, hidden test cases, and embedded submissions. Each submission includes: user, code, language, score, testCasesPassed, timeTakenSeconds, and aiAnalysis (timeComplexity, spaceComplexity, optimizationNotes, feedback).

Question Model (Question Bank)

Reusable coding questions with title, description, difficulty (Easy/Medium/Hard), category, tags, constraints, examples, test cases, and language-specific starter code templates (JS, Python, C++, Java).

Assignment Model

Multi-question assessment: references Question IDs, has deadline, time limit, max submissions per question. Submissions include correctnessScore, optimizationScore, plagiarismScore, and finalScore.

Scoring Formula: finalScore = correctness × 50% + optimization × 30% + originality × 20%

Quiz Model

MCQ quizzes with 4-option questions, time limits, single-attempt enforcement, and anti-cheat (tab-switching detection with 3-strike ban, banned user tracking).

7. Key Features

7.1 AI Code Evaluation Engine

  • Uses Groq SDK with LLaMA 3.3 70B model
  • Programmatic syntax pre-check: Validates brackets, strings, comments before AI
  • Correctness evaluation: AI traces code as a strict compiler against test cases
  • Optimization analysis: Time/space complexity, score (0-100), feedback
  • Plagiarism & AI detection: Compares against existing submissions, detects AI-generated patterns

7.2 Multi-Language Code Editor

  • Monaco Editor (VS Code engine) supporting JavaScript, Python, C++, Java
  • Language-specific starter templates, custom input testing, full test case evaluation

7.3 Quiz Anti-Cheat System

  • Full-screen enforcement, tab-switching detection (3-strike ban)
  • Server-side timer validation with 5-second grace period
  • Single attempt per quiz, unanswered questions excluded from scoring

7.4 Resume Evaluation

  • PDF upload (max 5MB), evaluates 7 categories (100 pts total)
  • Contact Info (15), Education (15), Experience (20), Skills (15), Formatting (15), Certs (10), Summary (10)
  • Letter grade (A+ to D) with actionable suggestions

7.5 Global Leaderboard

Aggregates best scores across all tasks, assignments, and quizzes per student. Sortable with breakdowns by category.

7.6 Company Analytics Dashboard

Overview metrics, score distribution, per-assignment breakdown, top performers, and recent submissions feed.

10. Setup & Installation

Prerequisites

  • Node.js v18+
  • MongoDB running locally on port 27017
  • Groq API Key for AI code evaluation

Backend

cd backend
npm install
# Configure .env (PORT, MONGO_URI, JWT_SECRET, GROQ_API_KEY, GEMINI_API_KEY)
node scripts/seedAdmin.js      # Create admin account (one-time)
node scripts/seedQuestions.js   # Populate question bank (one-time)
npm start                       # or: npm run dev (nodemon)

Frontend

cd frontend
npm install
npm start                       # Opens http://localhost:3000
Default Admin Credentials: Username: admin | Password: admin123

12. Security Features

  • Password Hashing: bcrypt with 10 salt rounds
  • JWT Authentication: Token-based auth with role claims
  • Role-Based Access Control: Three middleware guards
  • Company Approval Gate: Companies blocked until admin approval
  • Quiz Anti-Cheat: Tab-switching ban, full-screen, server-side timer
  • Submission Limits: Configurable max per task/assignment
  • File Upload Security: PDF-only, 5MB limit for resumes
